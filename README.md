# ✈️ White Cloud Airline Dashboard (Power BI Project)

📌 Project Overview

This project presents an interactive Airline Dashboard built using Power BI to analyze airline operations, passenger trends, and overall performance.

The dashboard provides actionable insights into key metrics such as flights, revenue, and passenger distribution, helping stakeholders make data-driven decisions.

🎯 Objectives
Monitor airline performance and operational efficiency
Analyze passenger trends and travel patterns
Track revenue and flight-related metrics
Provide an interactive and visually appealing reporting solution

🛠️ Tools & Technologies
Power BI
Data Visualization
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)

📊 Dashboard Features

1. ✈️ Flight Analysis
Total number of flights
Flight distribution across routes
On-time vs delayed flights

2. 👥 Passenger Insights
Total passengers
Passenger segmentation
Travel trends over time

3. 💰 Revenue Analysis
Total revenue generated
Revenue trends
Revenue by routes or regions

4. 🌍 Route & Geographic Insights
Popular routes
Country/region-wise performance
